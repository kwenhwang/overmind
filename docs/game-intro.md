# OVERMIND — 게임 소개서

> **적의 두뇌는 너를 관찰한다.**

## 한 줄 소개

플레이어의 습관을 실시간으로 관찰·학습하는 **진짜 AI(LLM) 보스**와 싸우는
웨이브 서바이벌 3D 아레나 액션. 같은 패턴을 반복하면 반드시 처벌당한다.

## 게임 개요

- 장르: 탑다운 3D 아레나 액션 (웨이브 서바이벌 + 보스전)
- 플랫폼: 웹 브라우저 (PC·모바일), 설치 불필요
- 플레이 타임: 한 판 7~10분
- 개발: 1인 (Claude Code 전면 활용) — 상세는 AI 활용 기술 문서 참조

### 무엇이 다른가

오버마인드(적 AI)는 각본이 아니라 **실제 LLM**이다. 웨이브 동안 당신의 회피 방향,
무기 선호, 위치 습관을 관찰하고 —

1. 웨이브 사이 **관찰 리포트**로 "본 것"을 공개하고 (회피 편향 ← 87% …)
2. 그 습관을 정조준한 **함정을 조합 설계**하며 (가시 드론 + 도주로에 가시 지대 …)
3. 왜 그렇게 설계했는지 **말로 밝힌다** ("넌 항상 왼쪽으로 구르더군.")

죽어도 끝나지 않는다 — 오버마인드는 **판을 넘어 당신을 기억**하고, 재도전하면
지난 판의 죽음을 언급하며 맞이한다. 웨이브 11개를 버티면 축적된 기록의 총결산:
당신에 대한 판결문을 낭독하고, 당신의 습관에 맞춰 설계된 페이즈로 직접 싸우는
**보스전**이 열린다.

## 게임 방법

### 목표
적 웨이브 11개를 버티고, 최종 보스 오버마인드를 파괴하라.

### 조작

| | PC | 모바일 |
|---|---|---|
| 이동 | `W` `A` `S` `D` / 방향키 | 왼쪽 화면 드래그 (가상 조이스틱) |
| 원거리 사격 | 마우스 **좌클릭** (또는 `K`/`J`) | 자동 |
| 대시 (무적) | 마우스 **우클릭** (또는 `Space`/`Shift`) | 오른쪽 화면 탭 |
| 근접 공격 | 밀착 시 **자동** 발동 | 자동 |
| 조준 | 마우스 | 자동 (최근접 표적) |

> 모바일은 조준·사격·근접이 전부 자동 — 이동과 대시(회피)에만 집중한다.

### 시스템

- **적**: 드론(예고 후 돌진), 스피터(원거리 포격), 브루트(저속 강타) — 모든 공격은
  예고(발광·경고 링) 후 발동되어 회피 가능하다.
- **모디파이어**: 오버마인드가 적에게 부착하는 강화 부품 — 가시(근접 반격), 정면 실드
  (등을 노려라), 분열, 자폭, 대시 미러(당신의 회피에 반응해 돌진), 원거리 가속.
- **해저드**: 가시 지대(지속 피해)·감속 지대 — 당신의 회피 방향을 봉쇄하도록 배치된다.
- **점수·랭킹**: 처치 콤보(3초 유지)와 웨이브 보너스로 점수 획득. 온라인 리더보드에 등록되며
  (버전별로 분리), 쉬운 조작(자동 조준·사격) 모드는 점수 배율이 낮다.

### 종료 조건

- **승리**: 보스 오버마인드 파괴
- **패배**: 체력 0 — 단, 오버마인드는 이 판에서 관찰한 것을 기억한 채 다음 판을 기다린다

## 실행 방법

1. 브라우저에서 접속: **https://kwenhwang.github.io/overmind/**
2. START — 별도 설치·로그인·API 키 불필요
3. 권장: 데스크톱 Chrome/Edge/Safari 최신, 사운드 ON. 모바일 완전 지원.
4. 소스 코드: https://github.com/kwenhwang/overmind (로컬 실행 방법은 README 참조)

> 네트워크가 차단된 환경에서도 게임은 동작한다 (LLM 프록시 불통 시 규칙기반 AI로
> 자동 전환 — 단, 오버마인드의 대사·설계가 단순해진다).
