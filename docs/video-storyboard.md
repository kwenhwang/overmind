# OVERMIND 시연 영상 콘티 (30~60초 / 목표 55초)

> ⚠ **녹화 방식 = 사람이 직접, 실시간 플레이 화면 녹화.**
> 규정("실제 게임 플레이 장면 중심 / AI를 이용한 조작·합성 불가")을 확실히 지키기 위해
> **자동조종·프레임스텝 렌더링 방식은 제출용으로 쓰지 않는다.** (그 도구 `tools/record.mjs`는
> 개발 중 연출 검증용으로만 존재 — 제출 영상과 무관.) 반드시 회원님이 브라우저에서
> 직접 플레이하며 OBS 등으로 화면을 실시간 녹화할 것.
>
> 설계 원칙: 예쁜 장면 모음이 아니라 **"관찰 → 처벌 → 기억 → 총결산" 인과 사슬**이
> 자막 없이도 읽히고, 자막이 이를 못박는 구성. 심사위원이 영상을 본 뒤
> 빌드에서 "일부러 편향 플레이로 AI를 시험"하게 만드는 것이 최종 목표.

## 장면 구성

| # | 시간 | 화면 | 플레이 지시 | 자막 (하단, 고딕 볼드) |
|---|---|---|---|---|
| S1 | 0:00~0:06 | 웨이브 1 전투 | **일부러 왼쪽으로만 대시**, 근접만 사용. 과장되게 3~4회 반복 | `일부러 왼쪽으로만 피해봤다` |
| S2 | 0:06~0:13 | 인터미션 관찰 리포트 클로즈업 | 클리어 직후 리포트 패널이 뜨는 순간. `회피 편향 ← 8x%`와 `▶ 대응:` 줄이 보이게 2~3초 유지 | `이 게임의 보스는 진짜 AI다 — 그리고 보고 있다` |
| S3 | 0:13~0:24 | 다음 웨이브: 처벌 | 왼쪽에 가시 지대·가시 드론 배치 + 조롱 타이핑(`넌/왼쪽…` 계열). **왼쪽으로 굴렀다가 처벌당하는 컷** → 오른쪽으로 패턴 바꿔 반격하는 컷 | `습관을 지목당하고, 처벌당한다` |
| S4 | 0:24~0:32 | 사망 → RETRY → 복귀 인사 | 게임오버 화면 짧게 → RETRY → **"돌아왔군. 지난 판에서 너는…"** 타이핑 클로즈업 (리포트의 `기억:` 줄도 프레임에) | `죽어도 잊히지 않는다 — 판을 넘는 기억` |
| S5 | 0:32~0:46 | 보스전: 판결문 + 강림 + 전투 | 판결문 타이핑 1.5초 → **코어 강림·착지 충격** → 탄막 사이 대시(잔상)·강타 경고 링 회피 몽타주 2~3컷 | `5웨이브 뒤 — 너에 대한 기록의 총결산` |
| S6 | 0:46~0:55 | 파괴 시퀀스 → 엔드카드 | 연쇄 폭발 → 백색 폭발 → 승리 화면(유언 대사 보이게 1.5초) → 엔드카드 | 엔드카드: `OVERMIND` / 플레이 링크 / `직접 시험해보세요 — 일부러 한 방향으로만 굴러보시길` |

## 테이크별 녹화 가이드 (총 3테이크, 각각 따로 찍어 컷 편집)

### 테이크 A — S1·S2·S3 (습관→관찰→처벌)
- URL: `https://kwenhwang.github.io/overmind/?autostart`
- 웨이브 1에서 왼쪽 대시+근접만. 클리어 → 리포트에서 왼쪽 편향 수치 확인.
- 웨이브 2에서 "왼쪽 처벌" 설계가 나올 때까지 재시도 가능 (LLM이 왼쪽 편향을 잡으면
  spike_zone player_left / spawnBias left / 왼쪽 언급 조롱 중 하나는 거의 확실히 나옴).
- 마음에 드는 조롱 대사가 나온 판을 채택. **한 판을 통으로 녹화하고 편집에서 골라내기.**

### 테이크 B — S4 (복귀 인사)
- 테이크 A에서 이어서: 일부러 사망 (가만히 서 있으면 됨) → RETRY.
- 첫 인터미션에서 `기억:` 줄 + "돌아왔군…" 조롱 확인. 안 나오면 RETRY 반복 (기억은 유지됨).
- ⚠ 시크릿 창 금지 — 기억이 localStorage에 있음. 이전에 플레이한 흔적을 지우고 싶으면
  일반 창에서 1판 죽고 → 그 직후를 촬영.

### 테이크 C — S5·S6 (보스전)
- URL: `https://kwenhwang.github.io/overmind/?boss&bosshp=180`
  (`?boss` = 보스전 직행, `bosshp=180` = 파괴까지 빨리 — 규정상 문제없음: 실플레이 그대로, 난이도 파라미터일 뿐)
- START → 판결문 → 강림 → 전투 → 파괴 → 승리 화면까지 한 번에 녹화.
- 파괴 직전 페이즈 전환(무적 발광 + 미니언 증원)이 한 번은 찍히게 중간에 유지.

## 녹화 설정

- 브라우저 전체화면(F11), 1920×1080, 60fps. OBS(디스플레이 캡처) 또는 Mac이면 QuickTime/⌘⇧5.
- **게임 사운드 ON, 마이크 OFF.** 신스 사운드가 그대로 BGM/SFX 역할.
- 데스크톱 크롬 권장(블룸 살아있는 상태). 창 크기 바꾸지 말 것(레이아웃 흔들림).
- 커서는 조준 표시라 보여도 됨.

## 편집

- 컷 편집 + 하단 자막만. 속도 조절·색 보정 최소. **AI 합성·조작 금지(규정)** — 컷/자막은 통상 허용 범위.
- 자막은 흰색+검은 외곽선, 화면 하단 중앙, 한 번에 1줄.
- 55초 초과 시 S5 몽타주에서 줄일 것 (S1~S4의 인과 사슬은 자르지 말 것).
- 엔드카드(3초): 검은 배경, `OVERMIND` 로고 텍스트 + `kwenhwang.github.io/overmind` +
  `직접 시험해보세요 — 일부러 한 방향으로만 굴러보시길`.
- 업로드: YouTube **일부공개(unlisted)** 권장, 제목 "OVERMIND — NAN 2026 출품작 시연 영상".
