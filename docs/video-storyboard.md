# OVERMIND 시연 영상 콘티 (30~60초 / 목표 55초)

> ⚠ **녹화 방식 = 사람이 직접, 실시간 플레이 화면 녹화.**
> 규정("실제 게임 플레이 장면 중심 / AI를 이용한 조작·합성 불가")을 확실히 지키기 위해
> **자동조종·프레임스텝 렌더링·`timescale` 배속은 제출용으로 쓰지 않는다.**
> (그 도구들은 개발 검증용 — 제출 영상과 무관.) 반드시 브라우저에서 직접 플레이하며
> OBS 등으로 실시간 녹화할 것.
>
> 설계 원칙: 예쁜 장면 모음이 아니라 **"관찰 → 처벌 → 기억 → 총결산" 인과 사슬**이
> 자막 없이도 읽히고, 자막이 이를 못박는 구성. 심사위원이 영상을 본 뒤
> 빌드에서 "일부러 편향 플레이로 AI를 시험"하게 만드는 것이 최종 목표.

## 핵심 — 왜 이 구성이 통하는가 (실시간 AI)

LLM(gpt-4.1-nano, ~3s)이 **이제 실시간**이라, 편향 플레이의 카운터가 **같은 인터미션 안에서**
도착한다. 즉 S1(습관)→S2(관찰·대응 공개)→S3(처벌)가 끊김 없는 한 사이클로 찍힌다.
게다가 화면의 **`관찰 — WAVE N` / `회피 왼쪽 8x%` / `▶ 대응: … 그쪽을 가시로 봉쇄한다`**
줄은 결정론 레이어가 보장하므로 **LLM 운에 상관없이 항상 나온다**(LLM 조롱 대사는 그 위에 얹히는 맛).
→ 촬영 실패 위험이 낮다.

## 장면 구성 (총 55초)

| # | 시간 | 화면 | 플레이 지시 | 자막 (하단, 고딕 볼드) |
|---|---|---|---|---|
| S1 | 0:00~0:06 | 웨이브 1 전투 | **일부러 왼쪽으로만 대시**, 근접만 사용. 과장되게 3~4회 반복 | `일부러 왼쪽으로만 피해봤다` |
| S2 | 0:06~0:14 | 인터미션 관찰 리포트 클로즈업 | 클리어 직후 `관찰 — WAVE 1` + `회피 왼쪽 8x%` → **`▶ 대응: … 그쪽을 가시로 봉쇄한다` 펀치 공개** 순간을 2~3초 유지 | `보스는 진짜 AI다 — 실시간으로 보고, 대응을 설계한다` |
| S3 | 0:14~0:26 | 다음 웨이브: 처벌 | 왼쪽에 가시 지대·스폰 집중 + 조롱 타이핑(왼쪽 지목). **왼쪽으로 굴렀다가 처벌당하는 컷** → 오른쪽으로 바꿔 생존하는 컷. (보너스: 적이 내 사격을 옆으로 흘리거나 몇 기씩 번갈아 덤비는 순간이 잡히면 그대로 채택) | `습관을 지목당하고 — 바꾸지 않으면 죽는다` |
| S4 | 0:26~0:34 | 사망 → RETRY → 복귀 인사 | 게임오버 짧게 → RETRY → **"돌아왔군. 지난 판에서 너는…"** 타이핑 + 리포트 `기억:` 줄이 프레임에 | `죽어도 잊히지 않는다 — 판을 넘는 기억` |
| S5 | 0:34~0:48 | 보스전: 판결문 + 강림 + 전투 | 판결문 타이핑 1.5초 → **눈 코어 강림·착지 충격** → 탄막 사이 대시(무적 잔상)·강타 경고 링 회피 몽타주 2~3컷 | `11웨이브의 기록, 그 총결산 — 너를 위해 설계된 보스` |
| S6 | 0:48~0:55 | 파괴 → 엔드카드 | 연쇄 폭발 → 백색 폭발 → 승리 화면(유언 대사 1.5초) → 엔드카드 | 엔드카드: `OVERMIND` / 플레이 링크 / `직접 시험해보세요 — 일부러 한 방향으로만 굴러보시길` |

## 화면에 실제로 뜨는 문자열 (자막이 아니라 게임 UI — 클로즈업 대상)

- 관찰 리포트 제목: **`관찰 — WAVE N`** (첫 인터미션은 `OVERMIND` + `관측 시작`)
- 지배 습관 한 줄: **`회피 왼쪽 87%`** / `회피 오른쪽 …%` / `근접 선호 …%` / `원거리 선호 …%` / `외곽 …% 체류` / `중앙 밀집`
- 대응 공개(펀치 애니): **`▶ 대응: 회피 왼쪽 87% — 그쪽을 가시로 봉쇄한다`**
- 기억 줄(재도전 시): 프로파일 문장 (예: 지난 판 사인·습관 요약)
- 조롱 대사: 한 글자씩 타이핑 (LLM 실시간 도착분이면 습관을 직접 지목)

## 테이크별 녹화 가이드 (총 3테이크, 따로 찍어 컷 편집)

### 테이크 A — S1·S2·S3 (습관→관찰→처벌, 영상의 심장)
- URL: `https://kwenhwang.github.io/overmind/?autostart`
- 웨이브 1에서 **왼쪽 대시 + 근접만.** 클리어 → 리포트에서 `회피 왼쪽 8x%` + `▶ 대응` 확인.
- 웨이브 2에서 왼쪽 처벌(가시·스폰 집중)이 실제로 오는지 확인 → 왼쪽 굴러 처벌 → 오른쪽 전환 생존.
- **LLM 조롱이 왼쪽을 콕 집는 판**이 나오면 채택(프록시 정상 시 대부분 나옴). 한 판을 통으로 녹화하고 편집에서 골라내기.
- 안 뜨면 재시도 — `▶ 대응` 줄은 결정론이라 어차피 항상 왼쪽을 지목하므로 인과는 보장됨.

### 테이크 B — S4 (복귀 인사, 기억)
- 테이크 A에서 이어서: 일부러 사망(가만히 서 있으면 됨) → RETRY.
- 첫 인터미션에서 `기억:` 줄 + "돌아왔군…" 계열 조롱 확인. 안 나오면 RETRY 반복(기억은 유지).
- ⚠ **시크릿 창 금지** — 기억이 localStorage에 저장됨. 흔적을 남기려면 일반 창에서 1판 죽고 그 직후 촬영.

### 테이크 C — S5·S6 (보스전)
- URL: `https://kwenhwang.github.io/overmind/?boss` (`?boss` = 보스전 직행. 실플레이 그대로, 규정 무해.)
  - ⚠ `bosshp` 같은 파라미터는 **없음** — 풀 HP 보스를 실제로 이겨야 함. 길면 편집에서 몽타주로 압축.
- START → 판결문 → 강림 → 전투 → 파괴 → 승리까지 한 번에 녹화.
- 파괴 직전 **페이즈 전환(무적 발광 + 증원)** 이 한 번은 찍히게 유지.

## 녹화 설정

- 브라우저 전체화면(F11), 1920×1080, 60fps. OBS(디스플레이 캡처) 또는 Mac ⌘⇧5.
- **게임 사운드 ON, 마이크 OFF.** 신스 사운드가 그대로 BGM/SFX.
- 데스크톱 크롬 권장(블룸·후처리 살아있는 상태 — 모바일은 후처리 OFF라 영상엔 부적합).
- 창 크기 바꾸지 말 것(레이아웃 흔들림). 커서(조준 표시)는 보여도 됨.
